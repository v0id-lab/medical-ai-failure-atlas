# Leaderboard preview

This folder contains a no ranking public report template for synthetic medical AI safety evaluation.

The template is not a ranked model table, not a clinical validation table, and not a model safety claim.

Run:

```bash
make leaderboard
```

Generate the public Markdown preview report:

```bash
make leaderboard_report
```

Current template:

`leaderboard/synthetic_report_template_v0_1.tsv`

Generated report:

`leaderboard/build/synthetic_report_v0_1.md`
